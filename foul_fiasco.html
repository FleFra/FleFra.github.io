About Foul Fiasco

"Foul Fiasco" is a 2D game developed in Unity for a school project where we had to create an arcade game. For this project, each team had different restrictions. Our team had to make a game for kids under 12 in greyscale, with the theme 'Sports'. We decided to make a game where you're running away from the referee of a football game, while simultaneously dodging obstacles and scoring goals for extra points. The game has simple mechanics and an online leaderboard.

Features:

  • Try to get the high score!
  • Retro pixel graphics!
  • Arcade-like controls!
  • Addicting gameplay!

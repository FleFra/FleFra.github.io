2048 in c# console

A console remake of a modern classic.

About 2048

"2048" is a classic and well-known puzzle game. For this project, I created a version of the game in C# for the console. This solo project was completed in just under a week, focusing on clean coding and functionality.

Features:

  • Remade entirely in C# for a console environment
  • Dynamic score tracking to challenge your high scores
  • Faithful recreation of the original game's mechanics

This project served as a great way to sharpen my skills in C# programming, particularly in implementing logical algorithms and user interaction in a text-based environment. Despite its simplicity, it was rewarding to recreate a game so widely recognized and enjoyed.

My Experience

This project was an excellent opportunity to deepen my understanding of C# programming and do everything myself instead of relying on teammates. I focused on creating clean, efficient code and implementing the logic required for the core mechanics of 2048. Reproducing the game's mechanics in a console environment challenged me to think critically about algorithms and user input handling without relying on graphical interfaces.

Throughout the process, I gained valuable experience in debugging, optimizing performance for text-based applications, and managing the structure of a solo project. Completing this project in under a week also helped me refine my time management and problem-solving skills. Overall, it was a rewarding experience to bring a widely recognized game to life in a new format.
